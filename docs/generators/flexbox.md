# Flexbox Configuration
*Last updated: 2026-06-10*

This document provides a technical overview of the `gen-flexbox` mixin and the corresponding flexbox utility map.

## Flexbox Generator Mixin

The `gen-flexbox($name)` mixin facilitates the generation of flexbox layout properties based on predefined configurations.

- **Parameter:** `$name`
  - This parameter corresponds to a key within the `$flexbox` map located in `src/map/flexbox`.
  - The mixin retrieves the associated CSS properties (e.g., `display: flex; flex-direction: column;`) to apply the requested layout.
  - An error is triggered if the specified key is not identified within the map.

## Implementation Examples

  @include mix.gen-flexbox('flex-center');

}
