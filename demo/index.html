Documentation

Lumina SASS is a modular, design-token-driven styling library built for modern web applications. It provides standardized colors, flexbox utilities, and powerful mixins to streamline your development process.

Flexbox-utility tools

Quick Usage

 
@use 'lumina-sass' as *

.container
  @include flex-center
  background-color: $deep-midnight-blue
  padding: 1rem
